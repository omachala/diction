## How to Contribute

### Bug Reports

Found a bug with the self-hosting setup? [Open an issue](https://github.com/omachala/diction/issues/new?template=bug_report.md) with:

- Your Docker and OS version
- The model(s) you're running
- Steps to reproduce the issue
- Relevant logs (`docker compose logs`)

### Feature Requests

Have an idea? [Open a feature request](https://github.com/omachala/diction/issues/new?template=feature_request.md). We especially welcome suggestions for:

- Self-hosting improvements
- Docker configuration options
- Documentation improvements
- New model support

### Pull Requests

1. Fork the repository
2. Create a feature branch (`git checkout -b my-feature`)
3. Make your changes
4. Test your changes with `docker compose up -d`
5. Submit a pull request

### Documentation

Documentation improvements are always welcome. If something is unclear in the self-hosting guide or README, please submit a PR.

## Code of Conduct

Be respectful and constructive. We're all here to make speech-to-text better.
